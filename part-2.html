Question-1: What are the different ways to select an element in the DOM?

Answer : বিভিন্ন পদ্ধতি রয়েছে যেমন
1. const element = document.getElementById("Id-name");
2. const elements = document.getElementsByClassName("Class-name");
3. const elements = document.getElementsByTagName("div");
4. const element = document.querySelector(".Class-name");
5. const elements = document.querySelectorAll(".Class-name");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : 1. innerHTML এর মাধ্যমে html এর ভিতরে থাকা সকল কোড কে পাওয়া যায় প্রয়োজনে এগুলো নিয়ে কাজ করা যায়
2. InnerText এর মাধ্যমে html tag এর ভিতরে থাকা সকল text কে পাওয়া যায় আবার চাইলে text add ও করা যায় বাদ ও দেওয়া যায়
3. Textcontent এর মাধ্যমে সকল লুকানো text পাওয়া যার কিন্তু html tag বাদে

Question-3: What is event delegation in the DOM?

Answer : Event deligation এর মাধমে একটি Parent Element উপর Event লিসেনার বাবহার করে বা যোগ করে এলিমিনেটর ইভেন্ট হ্যান্ডলিং বা নিয়ন্ত্রণ করা যায়

Question-4: What is event bubbling in the DOM?

Answer : Event bubbling হলো এক ধরনের ইভেন্টে বা ইভেন্টের উপর কাজ করা এই যেমন মাউস ক্লিক করলে একটা কাজ করবে বা ইভেন্ট ঘটবে কীবোড প্রেস করলে একটা কাজ করবে এ রকম বিভিন্ন কাজ করা যায়
